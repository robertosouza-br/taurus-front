Política de Privacidade

1. Introdução

A CALTER Construtora está comprometida com a proteção da privacidade e dos dados pessoais de seus usuários e clientes, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

2. Dados Coletados

O Sistema Taurus coleta e processa os seguintes tipos de dados:

  • Dados de Usuários: CPF, nome completo, email, telefone, cargo e permissões de acesso
  • Dados de Clientes: Nome, CPF, RG, endereço, telefone, email, dados financeiros
  • Dados de Imóveis: Informações sobre unidades, valores, características
  • Dados de Transações: Reservas, contratos, pagamentos, histórico de negociações
  • Dados de Uso: Logs de acesso, ações realizadas no sistema, timestamps

3. Finalidade do Tratamento

Os dados pessoais são processados para as seguintes finalidades:

  • Gerenciamento de vendas e reservas de imóveis
  • Controle de acesso e autenticação de usuários
  • Comunicação com clientes e prospects
  • Geração de relatórios gerenciais e comerciais
  • Cumprimento de obrigações legais e contratuais
  • Melhoria da qualidade dos serviços prestados
  • Auditoria e segurança da informação

4. Base Legal

O tratamento de dados pessoais é realizado com base nas seguintes hipóteses legais da LGPD:

  • Execução de contrato: Dados necessários para cumprimento de contratos de compra e venda
  • Obrigação legal: Cumprimento de obrigações fiscais, trabalhistas e regulatórias
  • Legítimo interesse: Gestão interna, segurança e melhoria de serviços
  • Consentimento: Quando aplicável, para finalidades específicas informadas

5. Compartilhamento de Dados

Os dados pessoais podem ser compartilhados com:

  • Parceiros comerciais: Bancos, financeiras e instituições de crédito para análise de financiamento
  • Prestadores de serviço: Empresas de TI, auditoria e consultoria, sob contratos de confidencialidade
  • Autoridades: Quando exigido por lei ou ordem judicial
  • Grupo Empresarial: Empresas do mesmo grupo econômico para fins administrativos

O compartilhamento é sempre limitado ao necessário e realizado com garantias adequadas de proteção.

6. Segurança dos Dados

A CALTER Construtora implementa medidas técnicas e organizacionais para proteger os dados pessoais:

  • Criptografia de dados sensíveis em trânsito e em repouso
  • Controle de acesso baseado em perfis e permissões
  • Autenticação forte com tokens e sessões controladas
  • Backups regulares e planos de recuperação de desastres
  • Monitoramento de segurança e logs de auditoria
  • Treinamento regular da equipe sobre proteção de dados
  • Testes periódicos de segurança e vulnerabilidades

7. Retenção de Dados

Os dados pessoais são retidos pelo tempo necessário para:

  • Cumprimento das finalidades para as quais foram coletados
  • Atendimento a obrigações legais (prazo de 5 anos para dados fiscais e contratuais)
  • Exercício regular de direitos em processos judiciais, administrativos ou arbitrais

Após o término do período de retenção, os dados são eliminados ou anonimizados de forma segura.

8. Direitos dos Titulares (LGPD)

Em conformidade com a LGPD, você tem os seguintes direitos sobre seus dados pessoais:

  • Confirmar a existência de tratamento de dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Anonimizar, bloquear ou eliminar dados desnecessários ou excessivos
  • Solicitar portabilidade dos dados
  • Eliminar dados tratados com seu consentimento
  • Revogar consentimento
  • Opor-se ao tratamento de dados

10. Cookies e Tecnologias Similares

O sistema utiliza tokens de autenticação armazenados localmente no navegador para manter sua sessão ativa. Estes dados são utilizados apenas para fins de autenticação e não rastreiam sua navegação.

11. Alterações na Política

Esta política pode ser atualizada periodicamente. Recomendamos a revisão regular. Alterações significativas serão comunicadas por email ou notificação no sistema.
