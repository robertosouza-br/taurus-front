Este é um sistema de autenticação e controle de acesso desenvolvido com Angular 17.

Recursos implementados:

  • Autenticação JWT com Spring Boot
  • Controle de roles e permissões
  • Guards para proteção de rotas
  • Interceptors para requisições HTTP
  • Layout responsivo com PrimeNG
  • Menu lateral expansível
